# ArmadaSpace
## 장르
- 2D SF 전략 TCG
## 게임 소개
- 전장의 각 거점(행성)에 주어진 덱의 유닛(카드)들을 배치하여 자원을 모으고 상대의 거점을 공격해 상대의 메인 거점(행성)을 차지하는 것이 목적인 턴제 전략 게임입니다.
## 개발 목적

- 기획자와 기획서로 소통하며 협업
